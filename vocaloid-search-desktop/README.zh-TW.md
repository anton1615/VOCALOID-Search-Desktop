# VOCALOID Search Desktop

一款用於搜尋 Niconico VOCALOID 影片的桌面應用程式。

## 版本資訊

### v1.5.0 (2025-03-12)

#### 播放列表狀態同步修復

- **修復「斷層」問題**：修復在搜尋、歷史紀錄、稍後觀看之間切換後，載入更多結果時第 51 筆影片出現排序異常的問題
- **播放狀態獨立**：播放中的影片不會因為切換頁面而中斷，直到您從其他列表選擇新影片
- **版本控制機制**：新增列表上下文版本控制，防止並行載入請求造成資料混亂
- **向後兼容**：保留舊版欄位以確保平滑升級

#### 技術改進

- 重構 Rust 狀態管理，採用 `ListContext` 和 `ActivePlayback` 模型
- 前端還原邏輯現在從 `list_context` 讀取狀態，確保資料一致性
- 新增版本預留機制，使並行的 `load_more` 請求失效

## 功能特色

- 進階篩選搜尋（觀看數、收藏數、留言數、喜歡數、上傳日期）
- 自訂公式排序與篩選
- 子母畫面（PiP）模式，邊做事邊看影片
- 觀看紀錄追蹤
- 深色/淺色主題
- 多語言支援（繁體中文、日文、英文）

## 安裝方式

### 隨身版（USB 隨身碟）

1. 下載 `vocaloid-search-desktop.exe` 執行檔
2. 在執行檔同目錄建立 `data` 資料夾
3. 執行程式，所有資料將儲存在 `data` 資料夾中

```
/your-usb-drive/
├── vocaloid-search-desktop.exe
└── data/
    ├── data.db
    └── config.json
```

### 標準安裝

1. 下載並執行安裝程式
2. 資料將儲存在 `%APPDATA%/vocaloid-search-desktop/`

## 系統需求

- Windows 10 或更新版本
- WebView2 Runtime（通常已內建於 Windows 10/11）

若未安裝 WebView2，程式會自動提示安裝。

## 使用說明

### 首次啟動

1. 程式會偵測到空白資料庫並導向同步頁面
2. 設定搜尋關鍵字（預設：「VOCALOID」）
3. 點擊「開始同步」從 Niconico 下載影片資料
4. 等待同步完成（大量資料可能需要數小時）

### 搜尋功能

- 在搜尋列輸入關鍵字後按 Enter
- 使用篩選列依觀看數、收藏數、留言數、喜歡數或上傳日期排序
- 點擊齒輪圖示開啟進階篩選
- 勾選「隱藏已觀看」排除已看過的影片

### 播放功能

- 點擊影片縮圖在內嵌播放器中播放
- 使用 PiP 按鈕開啟獨立的子母畫面視窗
- 開啟自動播放可自動播放下一部影片

### 自訂公式

依權重計算綜合分數：
- 分數 = (觀看權重 × 觀看數) + (收藏權重 × 收藏數) + (留言權重 × 留言數) + (喜歡權重 × 喜歡數)

## 資料庫更新提醒

程式啟動時會檢查資料庫新鮮度：
- Niconico Snapshot API 每日約 05:00 JST 更新
- 若資料庫更新時間早於今日 06:00，將提示同步

## 快速鍵

- `Enter` - 搜尋
- `Space` - 播放/暫停（播放器焦點時）
- `←` / `→` - 上一部/下一部影片

## 技術細節

- 以 Tauri 2.x 建構（Rust 後端 + Vue 3 前端）
- SQLite 資料庫搭配 FTS5 全文搜尋
- 使用 Niconico Snapshot API 取得影片資料

## 開發

```bash
# 安裝相依套件
npm install

# 開發模式執行
npm run tauri dev

# 建置正式版
npm run tauri build -- --no-bundle
```
